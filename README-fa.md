✨ تاریخچه تغییرات نسخه 1.5.2:
- **بازطراحی رابط کاربری:** طراحی مجدد اساسی پنل و کدهای ظاهری.
- **تغییر مسیر ورودی:** تعیین آدرس تغییر مسیر سفارشی (مثلاً https://google.com) برای درخواست‌های نامرتبط.
- **آدرس استتار (پروکسی معکوس):** استفاده از آدرس پروکسی معکوس به‌عنوان استتار.
- **نام فایل اشتراک:** امکان تغییر نام فایل اشتراک.
- **پیشوند پنل ادمین:** تعیین مسیر مخفی ادمین برای پنل.
- **دامنه‌های مسدود:** تعیین دامنه‌های مسدود (هر خط یک دامنه).
- **لینک‌های اشتراک کلش و سینگ‌باکس:** تولید لینک اشتراک برای کلاینت‌های Clash و Sing-box.
- **سازگاری با موبایل و تبلت:** بهبود واکنش‌گرایی برای صفحات موبایل و تبلت.
- **دقت لاگ‌ها:** افزایش دقت لاگ‌های ورودی/خروجی و داده‌های مصرف کاربران.
- **تقویت سیستم ضد خواب:** تقویت اساسی سیستم Keep-Alive.
- **لینک DOH:** افزودن لینک DNS-over-HTTPS (در برخی سرویس‌ها کار می‌کند).
- **پروفایل‌های آی‌پی:** اختصاص پروفایل‌های IP ویژه برای هر inbound.

✨ تاریخچه تغییرات نسخه 1.5.0 (بتا):
- **پروتکل XHTTP:** انتقال کامل SplitHTTP با حالت‌های packet‑up، stream‑up و stream‑one.
- **دروازه سهمیه تطبیقی:** بررسی‌های پهنای باند را بر اساس سرعت لحظه‌ای به‌صورت دسته‌ای انجام می‌دهد و فشار پایگاه داده را کاهش می‌دهد.
- **پاسخ خام Downlink:** پاسخ‌های GET بدون کدگذاری تکه‌تکه (chunked) برای سازگاری کامل با Xray‑core.
- **مسیریاب پویای XHTTP:** ارسال خودکار درخواست با استفاده از مسیر سفارشی inbound – بدون نیاز به پیکربندی اضافی.
- **تزریق خودکار ALPN:** در صورت عدم تعیین ALPN، به‌طور خودکار `alpn=http/1.1` را به لینک‌های XHTTP اضافه می‌کند.
- **ابزار عیب‌یابی XHTTP:** اسکریپت مستقل (`xhttp_diag.py`) که یک کلاینت را شبیه‌سازی کرده و تأخیر و توان عملیاتی را می‌سنجد.
- **بهبود امنیتی:** نقطه پایانی WebSocket اسکنر IP عمومی به‌طور پیش‌فرض غیرفعال شده است تا از سوءاستفاده جلوگیری شود.
- **بهینه‌سازی عملکرد:** افزودن `uvloop`، به‌روزرسانی Dockerfile و تنظیمات پلتفرم برای اجرای روان‌تر.

✨ تاریخچه تغییرات نسخه 1.1.0:
- بخش **UI و UX:** پالایش رابط شیشه‌ای (glass‑morphism)، رفع باگ تم آبی، بهبود واکنش‌گرایی موبایل.
- **عملکرد:** پاکسازی دوره‌ای حافظه نهان لینک‌ها، لغو صحیح وظایف اسکنر هنگام بستن WebSocket.
- **ضد خواب:** بازطراحی موتور Keep‑Alive با دو حالت (ساده/پیشرفته) که قابل تعویض در لحظه از پنل است.
- تغییرات **Inboundها:** پشتیبانی از Fragment (FRAG) برای دور زدن DPI؛ امکان اختصاص پرچم کشور به هر inbound.
- **داشبورد کاربر:** نوار پیشرفت مصرف زنده با آستانه‌های رنگی (سبز ← زرد ← قرمز) تا کاربران مصرف خود را فوری ببینند.
- **تلگرام:** رفع مشکل تغییر زبان (انگلیسی/فارسی) که اکنون به درستی ذخیره و بازیابی می‌شود.
- **پایگاه داده:** انتقال خودکار طرحواره – نصب‌های فعلی به‌طور خودکار ستون‌های `flag` و `fragment` را بدون دخالت دستی اضافه می‌کنند.
- **رفع باگ‌ها:** کارت‌های وضعیت تنظیمات اکنون با پیکربندی واقعی همگام‌اند، انتخاب‌گرهای منطقه زمانی و زبان کاملاً هماهنگ شده‌اند.
